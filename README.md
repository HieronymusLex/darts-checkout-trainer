# Darts Checkout Trainer

A web app for learning and practicing darts checkout routes.

## Features

**Lookup Mode** - Search for checkout routes by score. View recommended routes for 1, 2, and 3 darts in hand. Customize routes to match your preferred strategy.

**Test Mode** - Practice identifying correct checkout routes. Configure score ranges and darts in hand. Track your accuracy and average response time.

**Stats Mode** - Analyze which doubles you need to practice based on your checkout routes. View distribution across 2-dart, 3-dart, or combined finishes.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```
